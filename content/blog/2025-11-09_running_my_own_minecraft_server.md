For months, if not years, creating a **homelab** was on my mind, but I kept postponing it because I lacked a clear goal for what I'd run there.

When I finally started working on it in October, the motivation was very simple: My oldest kid wanted to play **Minecraft** together! :)

It took a bit of time and a lot of extra steps to ensure I did it the way I wanted. I learned so much about **Kubernetes** and all the related tools along the way, which was part of the goal.

I'm happy to share I now have a **Minecraft server** running in my **K8s homelab cluster**. It is always on, there is no lag, and it's running surprisingly well.

I have other lab projects in mind, but I am very happy that the idea that started it all is now complete.
