For many years, I've had scripts I'd run on a new **MacBook** to bring it up to the state I need with all the apps and settings I want. But for some reason, only recently I realized I need this for all of the operating systems I use.

Playing with VMs lately made it even more useful as I switch between **Mac**, **Windows**, and **Linux**. With Mac and Linux, everything was pretty easy. It's with Windows is where I really appreciated **AI**, as I don't have the time or patience to dive into **PowerShell**.

And now I have a [repository](https://github.com/prymost/os_bootstrap) I can easily pull and set up a brand new OS with everything I need and the way I like it. These are quite personalized, but if anyone wants to, you can definitely use it as inspiration.
